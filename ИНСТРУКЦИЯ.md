# ⚽ AI Спортивный Аналитик — Телеграм Бот (БЕСПЛАТНО)

## Что нужно: 2 бесплатных ключа

---

## 🔑 ШАГ 1 — Создай бота в Телеграме

1. Открой Телеграм → найди **@BotFather**
2. Напиши `/newbot`
3. Введи имя: например `Футбольный Аналитик`
4. Введи username: например `my_football_bot`
5. Получишь токен вида: `7123456789:AAHxxxxxxxxxxxxxx`
6. Сохрани его!

---

## 🔑 ШАГ 2 — Получи бесплатный Gemini API Key

1. Зайди на **aistudio.google.com**
2. Войди через Google аккаунт
3. Нажми **"Get API Key"** → **"Create API Key"**
4. Скопируй ключ вида: `AIzaSyXXXXXXXXXXXXXXXXXX`
5. **БЕСПЛАТНО**: 1500 запросов в день, 1 запрос в секунду

---

## 🚀 ШАГ 3 — Запусти на Railway (бесплатно)

1. Зайди на **github.com** → создай новый репозиторий (назови `football-bot`)
2. Загрузи в него все файлы из этой папки (bot.py, requirements.txt, Procfile)
3. Зайди на **railway.app** → войди через GitHub
4. Нажми **New Project** → **Deploy from GitHub** → выбери свой репозиторий
5. Перейди в **Variables** и добавь две переменные:
   - `TELEGRAM_BOT_TOKEN` = твой токен от BotFather
   - `GEMINI_API_KEY` = твой ключ от Google AI Studio
6. Railway автоматически запустит бота!

---

## 💬 Как пользоваться

1. Открой своего бота в Телеграме
2. Напиши `/start`
3. Напиши матч:
   `Реал Мадрид vs Барселона`
4. Жди 20-40 секунд → получи анализ!

---

## 📊 Что анализирует бот

- ⚽ Победитель матча + % уверенности
- 📈 Фора + % уверенности  
- 🔢 Тотал голов (больше/меньше) + % уверенности
- ⏱ Голы во 2-м тайме + % уверенности
- 🟨 Жёлтые карточки + % уверенности
- 🚩 Угловые + % уверенности
- 🏆 ГЛАВНАЯ СТАВКА (одна самая уверенная)

---

## ❓ Частые вопросы

**Бот не отвечает?**
→ Проверь что оба ключа правильно вставлены в Railway Variables

**Ошибка 429?**
→ Превышен лимит Gemini (1500/день). Подожди до следующего дня.

**Долго думает?**
→ Это нормально, бот ищет статистику через Google. Жди 30-60 сек.
